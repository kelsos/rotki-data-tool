# rotki-data-tool

A small python cli utility to help manage rotki data/accounts during development
## Installation

```bash
python -m venv .venv
. .venv/bin/activate
pip install --upgrade pip
pip install -r requirements.txt
```

## Usage

To see what the utility can do run the following.

```bash
./tool --help
```
